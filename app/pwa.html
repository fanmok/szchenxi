PWA帮助文档

by fanmok

如何安装PWA？

方法一：部分浏览器会自动弹出提示，点击“添加”即可

方法二：在浏览器上前往 找到 按钮，点击安装即可（一定要加载完毕）（对Safari无效）

如果在支持的浏览器上使用此方法弹出“浏览器可能不支持PWA”，请使用方法三

方法三：点击“...”，找到“添加到主屏幕”或“安装‘晨曦’”，点击“添加”即可

另外，请注意有没有给浏览器开启“创建桌面快捷方式”的权限

我的设备支不支持PWA？

现在流行的设备都支持安装，但是需要安装支持PWA的浏览器

想要知道自己的浏览器支不支持PWA？点击主页的按钮即可。

什么是PWA？

Progressive Web App, 简称 PWA，是提升 Web App 的体验的一种新方法，能给用户原生应用的体验。

PWA 能做到原生应用的体验不是靠特指某一项技术，而是经过应用一些新技术进行改进，在安全、性能和体验三个方面都有很大提升，PWA 本质上是 Web App，借助一些新技术也具备了 Native App 的一些特性，兼具 Web App 和 Native App 的优点。

PWA 的主要特点包括下面三点：
可靠 - 即使在不稳定的网络环境下，也能瞬间加载并展现
体验 - 快速响应，并且有平滑的动画响应用户的操作
粘性 - 像设备上的原生应用，具有沉浸式的用户体验，用户可以添加到桌面

PWA 本身强调渐进式，并不要求一次性达到安全、性能和体验上的所有要求，开发者可以通过 PWA Checklist 查看现有的特征。

附注

Windows的Microsoft Edge并不是指现在Windows 10自带的那个，而是Chromium内核的Edge

Google和Microsoft一直在推动PWA生态发展，所以它们两家的浏览器对PWA的兼容性都比较好（个人强推Chrome浏览器，浏览速度秒杀3*0、*讯）
Apple对PWA不太支持，即使Safari支持PWA，在使用方面也不太友好。
国内的生态确实对PWA不太友好，但是微信小程序和快应用其实跟PWA大同小异，也可以说PWA是它们的祖宗。
